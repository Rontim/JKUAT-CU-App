# JKUAT CU App

Welcome to the **JKUAT Christian Union App**, a mobile application designed to connect students of Jomo Kenyatta University of Agriculture and Technology (JKUAT) with their Christian Union community. The app helps members stay informed, engage with spiritual resources, and participate in CU activities.

## Project Description

The JKUAT CU App is a comprehensive tool designed to enhance the spiritual journey of CU members by providing:

- **Daily Verses**: Receive and reflect on daily Bible verses to enrich your faith.
- **Upcoming Events**: Stay updated with CU events, meetings, and gatherings.
- **Sermon Archive**: Access sermons from previous services and events, along with praise and worship archives.
- **Profile Management**: Manage your personal profile and connect with other CU members.
- **Giving & Funds Drive**: Support the CU through a dedicated section for offerings and contributions to ongoing funds drives.
- **Prayer Requests**: Share prayer requests and pray for fellow CU members.

Built using **Flutter** for cross-platform support, this app ensures that users on both Android and iOS can access these features seamlessly.

---

## Features

- **Daily Verses**: Get enriched with handpicked Bible verses daily to uplift and guide you.
- **Upcoming Events**: Browse and RSVP to events, and receive notifications on new CU activities.
- **Sermon Archive**: Listen to sermons from past services, access worship session recordings, and reflect on praise and worship archives.
- **Profile Management**: Edit and maintain your user profile, making it easier to stay connected with fellow CU members.
- **Giving & Funds Drive Section**: Contribute to the CU or specific causes through a secure giving platform.
- **Prayer Requests**: Share and respond to prayer requests within the Christian Union community.
- **Notifications**: Stay updated with real-time notifications on new activities, prayer requests, and events.

## Getting Started

### Prerequisites

Ensure that you have the following tools installed on your local machine:

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK (included with Flutter)
- Android Studio or Xcode (for iOS)
- A code editor (e.g., VS Code, IntelliJ IDEA)

---

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/JKUAT-CU-App.git
    ```
2. Navigate to the project directory:
   ```bash
    cd JKUAT-CU-App
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
